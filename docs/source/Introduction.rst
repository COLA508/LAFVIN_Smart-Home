螺丝尺寸比较
------------
  本产品在安装过程中会用到多种不同型号的螺丝。请务必在安装前逐一核对螺丝型号并参照本教程中对应步骤使用指定型号，
  严禁混用或以其他型号代替，以免因螺丝规格不符导致无法安装或损坏产品。如遇无法辨认，请停止操作并参阅配件清单或联系售后确认。
